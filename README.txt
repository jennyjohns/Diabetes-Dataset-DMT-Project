This repository consists of all the research/tools used within this Data Mining project on a diabetes dataset.
The repository includes a folder for all the code, which has a separate README file to explain how to run the code.
It also contains a folder which has all the figures from the exploratory analysis.
It contains a folder for all the proposals and updates created, as well as a folder containing the videos 
and presentation slides made for the project.

The project was completed on a diabetic dataset which can be found at the following link: 
	http://archive.ics.uci.edu/ml/datasets/Diabetes+130-US+hospitals+for+years+1999-2008#
